# Party Games!

The chaotic party game for 2-10+ players. Pass & Play or Multi-Device via room code.

## How to play

1. **Host** creates a game — choose Pass & Play (one phone) or Multi-Device (room code)
2. **Players** join — either add names on the host phone or enter the room code on their own devices
3. **Answer trivia** before the timer runs out — wrong answers or expiring timers lose a life
4. **Sabotage** opponents with curses (Blind Mode, Time Slice, etc.)
5. **Last player standing with the most points wins!**

Built with Next.js + Socket.io + Tailwind CSS + Framer Motion
